**==>Variables:**

&#x20;  It is used to store datatypes

&#x20;  **-->Good way to define:**

&#x20;     \*use underscore(\_)  EX: num=8

&#x20;     \*first letter should be small.  EX: num=7 (or) EX: nUM=7 

&#x20;     \*first letter should be capital.  EX: Num=5

&#x20;     \*if we want use variable with number use underscore.  EX: num\_1=4

&#x20;  -**->Bad way:**

&#x20;     \*NO special characters (@,$,%...etc)

&#x20;     \*Don't start with numbers(3num=4)

&#x20;     \*NO space B/W(n um=6)
